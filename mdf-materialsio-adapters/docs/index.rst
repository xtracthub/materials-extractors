Welcome to MDF MatIO Interface's documentation!
===============================================

The MDF MatIO library contains the interface needed for the
`Material Data Facility <https://www.materialsdatafacility.org/>`_
to use the `MaterialsIO Parser Library <https://materialsio.readthedocs.io/en/latest/>`_.
